# Google form抽選ツール ユーザーマニュアル

このマニュアルは、抽選ツールの基本的な使い方を簡単にまとめたものです。

## 1. 起動する

ターミナルで次を実行します。

```bash
python3 main.py
```

または:

```bash
python3 web_app.py
```

ブラウザでローカルページが開きます。

終了するときは、画面右上の「終了」または左側の「アプリを終了」を押してください。

## 2. ファイルを読み込む

左側のボタンを押します。

```text
CSV / Excel を選択して読み込み
```

CSV または Excel を選ぶと、自動で読み込まれます。

## 3. 抽選ID列を選ぶ

「列設定」画面で、抽選に使う ID 列を左クリックします。

選ばれた ID 列は青色になります。

例:

- 名前
- VRChat ID
- X ID
- Google Form の回答者名

## 4. 結果に表示する列を選ぶ

ID列を選んだあと、結果に表示したい列を左クリックします。

表示列は緑色になります。

例:

- X アカウント
- メモ
- 回答内容

もう一度クリックすると解除できます。

## 5. 特別条件を設定する

未選択の列を右クリックすると、特別条件を設定できます。

条件に合う応募者の確率を上げられます。

例:

```text
特別条件 = 【しました】 倍率 ×2
```

この場合、「しました」と回答した応募者の重みが 2 倍になります。

## 6. 応募者を除外する

応募者の行をクリックすると、その人を今回の抽選から除外できます。

除外された行は赤色になります。

もう一度クリックすると解除できます。

## 7. Event を選ぶ

左側の `Event` で、今回の抽選を保存する Event を選びます。

何も選ばない場合は `default` になります。

Event を作る・編集する場合は「Event編集」タブを使います。

## 8. 抽選人数を入力する

「抽選人数」に、当選させたい人数を入力します。

## 9. 確率モードを選ぶ

確率モードを選びます。

- `均等抽選`: 全員同じ確率
- `線形加重 - n^2`: 当たっていない人を少し優先
- `指数加重 - 2^n`: 当たっていない人を強く優先

当選した人は、次回の重みがリセットされます。

## 10. 抽選する

設定が終わったら「抽選開始」を押します。

抽選後、自動で「抽選結果」画面に移動します。

## 11. 結果を見る

「抽選結果」画面では、今回の当選者が上に表示されます。

下の「以前の抽選結果」から、過去の結果も見ることができます。

## 12. ユーザー一覧を見る

「ユーザー一覧」タブでは、保存されたユーザー履歴を確認できます。

Event を選ぶと、その Event の履歴だけを表示できます。

表示される内容:

- 抽選ID
- 参加回数
- 当選回数
- 重み
- 現在確率

## 13. よくある質問

### 抽選開始ボタンが押せません

CSV を読み込み、抽選ID列を選んでください。

### 行が赤くなっています

その応募者は除外されています。もう一度クリックすると解除できます。

### default とは何ですか？

Event を選ばなかった抽選の保存先です。

### 履歴を消したいです

アプリを終了してから、`vrc_raffle.db` を削除してください。

## 14. 基本の流れ

1. CSV / Excel を読み込む
2. 抽選ID列を選ぶ
3. 表示列を選ぶ
4. 必要なら特別条件を設定する
5. 必要なら応募者を除外する
6. Event を選ぶ
7. 抽選人数と確率モードを選ぶ
8. 抽選開始を押す
9. 結果を確認する
