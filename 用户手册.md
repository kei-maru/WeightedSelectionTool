# Google form抽選ツール ユーザーマニュアル

このマニュアルは、抽選ツールの基本的な使い方を簡単にまとめたものです。

## 1. 起動する

ターミナルで次を実行します。

```bash
python3 main.py
```

または:

```bash
python3 web_app.py
```

ブラウザでローカルページが開きます。

終了するときは、画面右上の「終了」または左側の「アプリを終了」を押してください。

## 2. ファイルを読み込む

左側のボタンを押します。

```text
CSV / Excel を選択して読み込み
```

CSV または Excel を選ぶと、自動で読み込まれます。

## 3. 抽選ID列を選ぶ

「列設定」画面で、抽選に使う ID 列を左クリックします。

選ばれた ID 列は青色になります。

例:

- 名前
- VRChat ID
- X ID
- Google Form の回答者名

## 4. 結果に表示する列を選ぶ

ID列を選んだあと、結果に表示したい列を左クリックします。

表示列は緑色になります。

例:

- X アカウント
- メモ
- 回答内容

もう一度クリックすると解除できます。

## 5. 特別条件を設定する

未選択の列を右クリックすると、特別条件を設定できます。

条件に合う応募者の確率を上げられます。

例:

```text
特別条件 = 【しました】 倍率 ×2
```

この場合、「しました」と回答した応募者の重みが 2 倍になります。

## 6. 応募者を除外する

応募者の行をクリックすると、その人を今回の抽選から除外できます。

除外された行は赤色になります。

もう一度クリックすると解除できます。

## 7. Event を選ぶ

左側の `Event` で、今回の抽選を保存する Event を選びます。

何も選ばない場合は `default` になります。

Event を作る・編集する場合は「Event編集」タブを使います。

## 8. 抽選人数を入力する

「抽選人数」に、当選させたい人数を入力します。

## 9. 確率モードを選ぶ

確率モードを選びます。

- `均等抽選`: 全員同じ確率
- `ゆるやか加重 - n + 1`: 当たっていない人を少し優先
- `二乗加重 - (n + 1)^2`: 当たっていない人を強く優先

当選した人は、次回の重みがリセットされます。

## 10. 抽選する

設定が終わったら「抽選開始」を押します。

抽選後、自動で「抽選結果」画面に移動します。

## 11. 結果を見る

「抽選結果」画面では、今回の当選者が上に表示されます。

下の「以前の抽選結果」から、過去の結果も見ることができます。

## 12. ユーザー一覧を見る

「ユーザー一覧」タブでは、保存されたユーザー履歴を確認できます。

Event を選ぶと、その Event の履歴だけを表示できます。

表示される内容:

- 抽選ID
- 参加回数
- 当選回数
- 重み
- 現在確率

右側の「Excel出力」を押すと、次の2つを保存できます。

- Sheet1: 今のユーザー確率
- Sheet2: 今までの抽選結果

## 13. 以前の履歴を同期する

ユーザー一覧の「データ同期」を押します。

1. CSV / Excel を選びます。
2. ID列をクリックします。青色になります。
3. 参加回数列をクリックします。紫色になります。
4. 当選回数列をクリックします。緑色になります。
5. 同期先 Event を選びます。
6. 「追加」または「上書き」を選びます。最初は「追加」になっています。
7. 「同期を実行」を押します。

`追加` は今の回数に、読み込んだ回数を足します。

`上書き` はその Event の今のユーザー一覧を消して、読み込んだファイルの内容に入れ替えます。

過去の当選日が分からないため、同期直後の重みは参加回数と当選回数から推定します。
`n = max(参加回数 - 当選回数, 0)` として、線形は `n + 1`、二乗は `(n + 1)^2` です。

同期を取り消す場合は、同期履歴の「元に戻す」を押します。
その Event の最新同期だけを元に戻せます。
同期前のユーザー一覧は丸ごと保存されるため、「元に戻す」で同期直前の一覧へ戻ります。

## 14. Docker で起動する

```bash
docker compose up -d --build
```

ブラウザで次を開きます。

```text
http://サーバーのIP:8765/
```

停止:

```bash
docker compose down
```

DB は `data/vrc_raffle.db` に保存されます。

## 15. よくある質問

### 抽選開始ボタンが押せません

CSV を読み込み、抽選ID列を選んでください。

### 行が赤くなっています

その応募者は除外されています。もう一度クリックすると解除できます。

### default とは何ですか？

Event を選ばなかった抽選の保存先です。

### 履歴を消したいです

アプリを終了してから、`vrc_raffle.db` を削除してください。

## 16. 基本の流れ

1. CSV / Excel を読み込む
2. 抽選ID列を選ぶ
3. 表示列を選ぶ
4. 必要なら特別条件を設定する
5. 必要なら応募者を除外する
6. Event を選ぶ
7. 抽選人数と確率モードを選ぶ
8. 抽選開始を押す
9. 結果を確認する
